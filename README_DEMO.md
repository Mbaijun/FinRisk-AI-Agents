# 🤖 FinRisk-AI-Agents 交互演示

这是一个 **FinRisk-AI-Agents** 核心功能的快速体验版。

## 🎯 体验内容
输入任意上市公司名称或股票代码，系统将自动模拟生成一份**金融风险与舆情简报**，展示多智能体协同分析的输出效果。

## 🚀 快速开始
1.  在左侧输入框填写公司名称（如 `Tesla`）。
2.  点击“提交”按钮。
3.  查看右侧生成的详细风险分析报告。
