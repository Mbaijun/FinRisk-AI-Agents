## 🚀 项目简介
**FinRisk-AI-Agents** 是一个开源、模块化的AI多智能体系统，旨在为金融风控与投资分析提供**自动化、可解释的智能洞察**。它模拟专业分析师团队，7x24小时监控多源数据，进行交叉验证与深度分析，解决信息过载的核心痛点。

> **🔍 稀少性 & 💰 商业潜力**：针对金融垂直领域的开源智能体解决方案，直接服务于高付费意愿的金融科技与投资机构。

## ✨ 核心功能
- **🤖 多智能体协同工作流**：全自动化处理数据采集、清洗、分析到报告的全流程。
- **🔍 信源交叉验证**：对关键风险信息进行多平台溯源与可信度评分，对抗虚假信息。
- **📈 可解释的分析报告**：提供附带证据链与推理路径的深度报告，而非黑盒结论。
- **⚙️ 模块化设计**：支持灵活添加新的数据源、分析规则或风险模型。

## ⚡ 快速开始
### 环境准备
```bash
# 1. 克隆本项目
git clone https://github.com/Mbaijun/FinRisk-AI-Agents.git
cd FinRisk-AI-Agents

# 2. 安装依赖 (请确保你已更新 requirements.txt)
pip install -r requirements.txt

# 3. 配置环境变量
cp .env.example .env  # 然后编辑 .env 文件，填入你的API密钥等

# 4. 运行示例
python run.py
