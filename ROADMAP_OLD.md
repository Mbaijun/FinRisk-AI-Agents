# FinRisk-AI-Agents 开发路线图

## 🎯 愿景
成为金融科技领域最受信赖的开源智能分析基础设施。

## 📅 近期开发计划 (Next 3 Months)
### 阶段一：MVP (最小可行产品) - 基础监控
- [ ] **数据管道**：集成至少2个公开数据源（如NewsAPI， Twitter API v2）。
- [ ] **智能体核心**：实现基础的数据处理与分析链（LangChain）。
- [ ] **输出**：生成标准化的JSON格式风险事件简报。
- [ ] **演示**：提供一个可交互的Streamlit/Gradio演示界面。

### 阶段二：增强验证 (核心价值)
- [ ] **验证智能体**：开发信源交叉验证逻辑与可信度评分。
- [ ] **报告生成**：从简报升级为结构化的HTML/PDF分析报告。
- [ ] **基础警报**：实现基于规则（关键词、情绪）的邮件/Webhook警报。
